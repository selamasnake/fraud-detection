# Notebooks Overview

This folder contains the analytical workflow for the fraud detection project.

## Notebooks
- `eda-fraud-data.ipynb`: Exploratory data analysis for e-commerce fraud data.
- `eda-creditcard.ipynb`: Exploratory data analysis for bank credit card transactions.
- `feature-engineering.ipynb`: Feature creation, encoding, scaling, and resampling.
- `modeling-fraud-data.ipynb`: Model training, evaluation, and selection for fraud data.
- `modeling-creditcard.ipynb`: Model training, evaluation, and selection for credit card data.
- `shap-explainability-fraud-data.ipynb`: Model interpretability using SHAP for fraud data.
- `shap-explainability-creditcard.ipynb`: Model interpretability using SHAP credit card data.
